# Klusterointi

Cluster network program. It makes a network of haplogroup clusters based GDs of tested person matches. It don't use
Gephi anymore. It uses networkx of python.

This program is not a ready project. It's on the way to finish. It is programmed with python and PyCharm.
Now it can draw picture of clustered haplogoup with MDKAs. But not from kitlists.

Input to this program are downloaded mt-dna match lists from FTDNA.
